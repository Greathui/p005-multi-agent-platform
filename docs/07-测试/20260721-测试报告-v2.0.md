### 测试中发现并修复的 Bug

| 编号 | 严重程度 | 描述 | 修复方式 |
|---|---|---|---|
| BUG-01 | **严重** | `msg.thinking_override` 属性不存在（ChatMessage 字段名是 `enable_thinking`），导致子智能体执行时 SSE 流中断 | 2 处 `msg.thinking_override` → `msg.enable_thinking` |
| BUG-02 | **严重** | `execute_tool` 中 invoke_sub_agent 调用引用了未定义的 `thinking_override` 变量 | `thinking_override=thinking_override` → `thinking_override=None` |

**这两个 bug 之前未被发现**，因为之前的测试都是 API 层测试，没有真正调用 LLM 触发多智能体协作。这正是 LLM 长任务测试的价值所在。
