## 文档维护约定

新增/修改文档时请遵守 [文档维护规范](./03-指导/20260719-文档维护规范.md) 和 [AGENTS.md 第十二节](../AGENTS.md)：

1. **单一权威来源**：每个知识点只能有一个权威文档，其他文档引用不复制。新建前先搜索是否已有同类文档，有则更新不新建
2. **命名规范**：活文档（长期维护）用语义化命名不带日期前缀，死文档（历史快照）带日期前缀（`YYYYMMDD-中文名.扩展名`）
3. 文档开头加状态头（📝草稿 / 📌有效 / 🗜️已归档）
4. 任何变更在 [CHANGELOG.md](./CHANGELOG.md) 的 `[未发布]` 节点登记
5. 删除/废弃功能时，旧文档状态改"已归档"，移到 `04-设计/已归档/`
6. **定期清理**：每次发版做轻量清理（归档被替代的旧版、校准本文件的实现状态表、转正版本号），不积压到末期
