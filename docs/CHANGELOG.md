## [v0.5.0] - 2026-07-22

### 新增
- **智能体记忆系统**：agent 新增 `memory_log` 字段，任务完成后自动提取关键经验，构建上下文时注入系统提示词。新增 3 个记忆管理 API
- **跨项目经验迁移**：团队级 `shared_memory`，项目记忆超 10 条自动上传，新项目自动继承
- **对话摘要压缩**：截断时生成被删消息摘要注入 system，避免早期决策丢失
- **调度轮次优化**：截断优先保留工具调用消息，CORE_RULES 新增工具使用防退化规则
- **TODO 前端可视化**：新增 `TodoPanel.vue` 看板组件 + API + 自动刷新机制
- **文件版本控制**：write/edit 前自动备份（保留 10 版），新增 `list_file_versions`/`file_rollback` 工具
- **智能体性能追踪**：agent 新增 `stats` 字段，任务完成自动更新统计
- **API 配置开关控制**：模型配置新增 `enabled` 字段，可独立启用/禁用
- **捐赠打赏页面**：设置面板新增"关于与支持"页面

### 修复
- 记忆提取模型配置缺失（`trigger_memory_extraction` 缺少 `agent` 参数）
- 幻觉检测正则误报（`task_result` 被误匹配为任务 ID）
- 多项目并行时消息并发写入丢失（新增 5 个原子消息函数）
- 后端事件循环阻塞 + 前端竞态条件（generate 改同步 + 前端事件缓存+重放）

### 文档
- GitHub 开源发布（AGPL-3.0 协议 + 双重许可模式）

## [v0.4.0] - 2026-07-20

### 新增
- **技能绑定到智能体**：`enabled_skills` 字段 + 工具过滤逻辑增强 + 技能提示词注入
- **子代理调用插件**：`invoke_sub_agent` 工具，非递归，最大 8 轮工具循环
- **蓝图应用技能分配**：`api_apply_blueprint` 读取 member.skills 和 can_invoke_sub_agent
- **元团队常驻化重构**：3 个常驻专家 + 三阶段引擎（方案撰写→评审→融合）+ 评审闭环 + 专家进化
- **团队主页（TeamHome）**：团队信息卡、成员管理、项目列表
- **团队→项目两级树结构**：侧边栏改为团队>项目两级导航
- **导出团队为蓝图**：`POST /api/conversations/{conv_id}/export-blueprint`
- **元团队 Phase 4 优化**：自动流转 + 超时处理 + 报告导出 + 提示词版本历史 + 诊断对比

### 变更
- 元团队从"项目类型"重构为"常驻功能模块"（侧边栏一级入口）
- 旧接口 `create_meta_team` 标记为废弃（返回 410 Gone）

### 修复
- 页面切换时项目丢失（SSE 未中止 + 消息仅存内存）
- 后台运行消息显示异常（事件缓存+重放方案替换引用交换）
- 团队操作按钮不可见（CSS class 冲突）
- 14 项 P1 问题修复（安全加固 + 数据一致性 + 功能完善 + 前端清理）
- 12 项 P0/P1 问题修复（原子写入 + 并发锁 + api_key 脱敏 + 权限校验等）
