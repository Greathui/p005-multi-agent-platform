## 通用约定

### 返回格式

| 场景 | 格式 |
|---|---|
| 列表 | `{"资源复数名": [...]}` |
| 单对象 | `{"资源单数名": {...}}` |
| 操作成功 | `{"status": "ok", "message": "..."}` |
| 错误 | HTTP 4xx/5xx + `{"detail": "错误信息"}` |
| 文件列表 | `{"items": [...], "path": "绝对路径"}` |

### API Key 脱敏

所有返回 `api_key` 的接口均返回 `前4***后4` 格式。前端编辑时若检测到 `***`，应剔除 payload 中的 `api_key` 字段。

### SSE 事件通用格式

```
event: 事件类型
data: {"agent_id": "...", "content": "...", ...}

```

所有 SSE 端点返回 `StreamingResponse(media_type="text/event-stream")`，带 `Cache-Control: no-cache`、`Connection: keep-alive`、`X-Accel-Buffering: no` 头。

## 五、聊天（3 个）

### POST `/api/chat/stream` — SSE 流式聊天（核心）

**Body**:
```json
{
  "agent_id": "main",
  "message": "用户消息",
  "conversation_id": "项目ID",
  "direct": false,
  "default_config_id": "模型配置ID",
  "enable_thinking": null
}
```

**SSE 事件**:

| 事件 | data 字段 | 说明 |
|---|---|---|
| `agent_start` | `{agent_id}` | 智能体开始回复 |
| `token` | `{agent_id, content}` | 流式 token |
| `tool_start` | `{agent_id, tool_name, tool_args}` | 工具调用开始 |
| `tool_end` | `{agent_id, tool_name, result}` | 工具调用完成 |
| `agent_end` | `{agent_id, full_content}` | 智能体回复完成 |
| `batch_start` | `{batch_id, agent_ids}` | 并行调度开始 |
| `batch_end` | `{batch_id}` | 并行调度结束 |
| `hallucination_warning` | `{agent_id, warning}` | 幻觉检测警告 |
| `error` | `{message}` | 错误 |
| `done` | `{}` | 流结束 |

### GET `/api/messages` — 获取消息历史

Query: `conversation_id`

返回: `{"messages": [...]}`

### POST `/api/chat/regenerate` — 重新生成

Body: `{conversation_id, message_id, direct?, target_agent_id?}`

返回: SSE 流（同 `/api/chat/stream`）

## 十二、文件操作（5 个）

所有文件操作都做权限校验：根据 `conversation_id` + `agent_id` 找到智能体，校验 `allowed_paths`。

| 方法 | 路径 | 参数 | 返回 |
|---|---|---|---|
| POST | `/api/files/list` | Body: `{agent_id, path, conversation_id?}` | `{"items":[{name,is_dir,path}], "path"}` |
| POST | `/api/files/read` | Body: `{agent_id, path, conversation_id?}` | `{"content":"...", "path"}` |
| GET | `/api/files/read-image` | Query: `path, conversation_id?, agent_id?` | `FileResponse`（png/jpg/gif/svg/webp） |
| POST | `/api/files/upload` | Form: `conversation_id, agent_id?, sub_dir?, file` | `{"status":"ok","path","relative_path","filename","size"}` |
| POST | `/api/files/write` | Body: `{agent_id, path, content, conversation_id?}` | `{"status":"ok","path"}` |

**上传限制**: 最大 10MB；禁止 .exe/.bat/.sh/.cmd/.ps1 等可执行扩展名。

## 附录：SSE 事件类型汇总

| 事件 | 触发场景 | data 字段 |
|---|---|---|
| `agent_start` | 智能体开始回复 | `{agent_id}` |
| `token` | LLM 流式 token | `{agent_id, content}` |
| `tool_start` | 工具调用开始 | `{agent_id, tool_name, tool_args}` |
| `tool_end` | 工具调用完成 | `{agent_id, tool_name, result}` |
| `agent_end` | 智能体回复完成 | `{agent_id, full_content}` |
| `batch_start` | 并行调度开始 | `{batch_id, agent_ids[]}` |
| `batch_end` | 并行调度结束 | `{batch_id}` |
| `hallucination_warning` | 幻觉检测警告 | `{agent_id, warning}` |
| `expert_start` | 元团队专家开始 | `{expert_id, expert_name}` |
| `expert_complete` | 元团队专家完成 | `{expert_id, content}` |
| `phase_complete` | 元团队阶段完成 | `{phase, stats}` |
| `fusion_start` | 融合阶段开始 | `{}` |
| `fusion_complete` | 融合完成 | `{content}` |
| `all_complete` | 全流程完成 | `{blueprint}` |
| `error` | 错误 | `{message}` |
| `done` | 流结束 | `{}` |
